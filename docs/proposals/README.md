## 新しい提案の追加方法

1. 提案書をMarkdown形式で作成
2. 提案日、概要、実装計画を明記
3. `/CLAUDE.md`から参照できるよう適切にリンク
4. 実装完了時はステータスを更新

## 注意事項

- 実装済みの提案書は歴史的記録として保持
- 新しい提案は既存のアーキテクチャとの整合性を確認
- 大きな変更の場合は必ず提案書を作成してからコード実装を開始
